# leanringSystematic
系统性的学习的笔记

看书学习观大局，重细节。按照书本划分的大章、小节一步一步学习，学完一部分总结一部分。

做笔记应该以一小节为单位，学完一小节，归纳总结这一小节的内容，整理成自己的语言。

> 做笔记切忌大段大段的抄书，效率太低，不利于下次翻阅查找。
>
> 毕竟细节的东西还是书上的详细。
>
> 笔记应该是导航式的卡片内容，以简练的语言归纳总结书中内容，方便自己以后快速定位。
>
> 看书的过程中遇到觉得重要的内容，多看几眼，记下来。
>
> 看书过程中遇到重复的、简单的内容，快速过一遍，甚至可以跳过。
>
> 看书学习不应该是背书、抄书！
