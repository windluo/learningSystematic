### 优先掌握

#### React技术栈

- React.js
- React Router
- Redux
- Ant Design

至少需要学完 React.js、React Router、Redux。Ant Design在用到的时候再去看官方文档即可。

#### JavaScript

- ~~javascript高级程序设计（第3版）~~
- Javascript权威指南
- You-Dont-Know-JS

至少要学完 高程3和权威指南。

#### Node

- Node.js开发指南
- 深入浅出Node.js

#### Http

- HTTP权威指南

#### CSS/CSS3

- CSS: The Missing Manual
- CSS权威指南
- CSS禅意花园

#### HTML/HTML5

- HTML & CSS设计与构建网站
- HTML5秘籍

最基本的要求是过一遍，了解有这么个东西，然后在实际的开发中运用并掌握精通。
