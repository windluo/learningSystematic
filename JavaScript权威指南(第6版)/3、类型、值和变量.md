## 类型、值和变量

本章共28页，预计1个小时

JS中的算术运算在溢出、下溢或被零整除时不会报错，会返回一个特殊的无穷值。

```js
Infinity // 会保留 ± 号
```

JS 有一个非数字值 `NaN `，这个值与任何值都不相等，包括它自己。

因为 JS 采用 IEEE 754标准定义浮点数和数字，所以会出现精度丢失问题。

```js
// 这个是标准的问题，JS 无法解决
0.1 + 0.2 === 0.3; // false。0.1 + 0.2实际计算结果为 0.30000000000000004
```

本章对一些基础的概念做了简单的介绍。
